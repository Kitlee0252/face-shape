## 技术方案

```
用户上传照片 / 实时摄像头
  → 图片预处理（调整大小、去噪）
  → MediaPipe Face Mesh 检测 468 个面部关键点
  → 计算比例（额头/颧骨/下颌宽度、脸长）
  → 分类为 6+ 种脸型（含混合型）
  → 输出结果 + 个性化建议（发型/眼镜/妆容）
```

技术栈：MediaPipe Face Mesh + TensorFlow.js（浏览器端运行，无需后端）
